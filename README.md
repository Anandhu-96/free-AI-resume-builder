## PERSONAL AI RESUME BUILDER

Project Description:
This project is a Personal AI Resume Builder that allows users to create and update their resume live using an interactive interface. The system provides a structured high-quality resume format and allows users to add details using both manual input and AI-style prompts.

The resume updates instantly and is automatically saved so that when the application is opened again, the previous resume data is loaded.

---

Features:

* Live resume preview
* AI-style prompt based resume update
* High quality professional resume format
* Automatic data saving
* Automatic data loading
* Simple and user-friendly interface
* Real-time resume editing
* Structured resume sections
* Lightweight local database (JSON)

---

Technologies Used:

* Python
* Streamlit
* JSON (for data storage)

---

Project Structure:
resume_ai.py → Main application file
resume_ai_data.json → Auto generated resume database file
README.txt → Project documentation

---

How to Run the Project:

Step 1: Install Required Library
Open terminal or command prompt and run:
pip install streamlit

Step 2: Run the Application
Navigate to project folder and run:
streamlit run resume_ai.py

Step 3: Open in Browser
The application will automatically open in your web browser.

---

How the System Works:

1. The user opens the application.
2. A professional resume format is displayed.
3. The system asks the user to enter details.
4. The resume updates instantly as the user adds data.
5. The resume is automatically saved.
6. When the user reopens the app, the same resume appears.
